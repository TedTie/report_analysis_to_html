# Terminology Confirmation

Business terms must be confirmed when they affect analysis, report wording, or external presentation.

## Rule

Do not silently replace terms. When a term may be ambiguous, list:

- Original term.
- Current understanding.
- Possible misunderstanding.
- Suggested display wording.
- Question for the user.

Use the confirmed wording consistently afterward.

## Terms Often Requiring Confirmation

- account / CRM account / person / staff / consultant / team member
- entry / registration / lead / assigned lead
- booking / appointment / reservation
- attendance / show-up / attended class
- paid / converted / closed
- market / referral / campaign / organic / old-bring-new
- productivity / human efficiency / average output
- headcount / active producer / employed staff
- complete month / cutoff date / staged data / estimated data
- target / actual / completion rate
- channel share / channel contribution

## Cutoff Periods

Only mark a period as partial when the data indicates a cutoff date, incomplete period, estimate, or stage snapshot. Do not assume a month is incomplete based only on its name.
